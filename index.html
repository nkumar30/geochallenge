🌍 GeoChallenge Setup

Connect your Firebase project for real-time cross-device play

1

Go to console.firebase.google.com → Create a project (or use existing)

2

Click Realtime Database → Create database → Start in test mode (you can lock it down later)

3

Go to Project Settings → Your apps → Add Web app → Copy the firebaseConfig object and paste it below

Your config is saved in this browser so you won't need to paste it again.
